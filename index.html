Lucerne

build modern web applications with Common Lisp

Lucerne is a web framework for Common Lisp. It is inspired by Flask, and built on top of Clack.

Lucerne is simple

Views are functions. Apps map URLs to functions. Requests are function calls, and responses are their result. That's it. 

          
;; Name this file `myapp.lisp`
(use-package :lucerne)      ;; Import the `lucerne` module
(annot:enable-annot-syntax) ;; Enable the decorator pattern

(defapp app)

@route app "/"
(defview index ()
  (respond "Hello, world!"))

@route app "/greet/:name"
(defview greet (name)
  (respond (format nil "Hello, ~A!" name)))
          
        
          
;; Now, from a Lisp REPL, we load Lucerne and then our app
cl-user> (ql:quickload :lucerne)
cl-user> (load "myapp.lisp")
cl-user> (lucerne:start 'myapp::app)
Lucerne is small

With less than 200 SLOC and just 15 exported symbols, Lucerne is small enough to be understood and used productively in no time.

Lucerne is flexible

It's not tied to an ORM or a web server. You can use {{ site.crane }} or {{ site.postmodern }} to talk to your database, {{ site.wookie }} for an asynchronous server or {{ site.hunchentoot }} for a traditional one.
